Add a Pause-Menu

Instead of creating a separate scene, the Pause-Menu is just a CanvasLayer of the Level-Scene. It is shown in Pause-mode and otherwise hidden.

Also the game-tree will be set to pause to stop processing of game-engine.

Create a Main-Menu

The Main-Menu is a Scene containing a CanvasLayers with Buttons to switch to the Level-Scene.



  • you need to adjust the main scene in Project->Project Settings-> Application->Run

Collision Handling

  • the CharacterBody has a CollsionShape that is used to interact with the Level-Obstacles

  • you might want to add additional Area2D to detect interaction with other parts of the game (sensor-ray, collecting items,...)

  • Note: overlapping Collsionshape of the SAME Node will cause the body_entered Signal to fire. Unless you configure the Mask & Layer of the area to filter it

  • if the root-node is Area2D instead of RigidBody2D, you have to use on area2d_shape_entered instead of body_entered
